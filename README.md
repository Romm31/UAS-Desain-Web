# UAS Desain Web - SinergiKita

Project frontend ini dibuat untuk memenuhi tugas **Ujian Akhir Semester (UAS)** mata kuliah Desain Web. Website ini merupakan platform layanan sosial masyarakat bernama **SinergiKita**.

## ï¸ Tech Stack

Project ini dibangun menggunakan teknologi berikut:

- **HTML5**
- **Bootstrap 5** (via CDN) - Framework CSS untuk layout yang responsif.
- **Alpine.js** (via CDN) - Framework JavaScript ringan untuk interaktivitas.
- **Bootstrap Icons** - Ikon vektor yang digunakan dalam antarmuka.
- **Google Fonts** - Menggunakan font 'Inter' untuk typografi.
